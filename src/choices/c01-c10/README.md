# คำถามที่ 02 (C02)

**Input:** `[1,2,3,4]`  
**Output:** `[2,4]`

การดำเนินการเชิงแนวคิดใดอธิบายวิธีได้ผลลัพธ์นี้ได้ดีที่สุด (เก็บเฉพาะตัวเลขคู่)?

- **A.** แปลงทุกองค์ประกอบ (ความยาวเท่ากัน)
- **B.** เลือกเฉพาะองค์ประกอบที่ตรงตามเงื่อนไข
- **C.** ลดลงเป็นค่าเดียว
- **D.** คืนค่า subarray แบบ shallow ตามตำแหน่ง
